# LIFE-LINE-TECH---REAL-TIME-MEDICAL-AVAILABILITY-TRACKER
Lifeline Tech is a real-time medical availability tracking web application that helps users find nearby pharmacies with required medicines using GPS and live data. It features an interactive map, emergency mode, and an admin panel for managing inventory, improving healthcare accessibility and response time.

# 🏥 Lifeline Tech – Real-Time Medical Availability Tracker

A smart healthcare web application that helps users find **real-time medicine availability** in nearby pharmacies using **GPS, maps, and live inventory updates**.

---

## 📌 Overview

Lifeline Tech is designed to solve a critical healthcare problem — **difficulty in finding medicines during emergencies**.

Instead of visiting multiple pharmacies, users can instantly search and locate medicines nearby with accurate availability information.

📄 Based on project report and system design 

---

## 🚨 Problem Statement

* No centralized system for medicine availability
* Time wasted visiting multiple pharmacies
* Lack of real-time stock updates
* Difficulty locating nearby pharmacies

📊 As highlighted in project presentation slides 

---

## 💡 Solution

Lifeline Tech provides:

* 🔍 Real-time medicine search
* 📍 GPS-based location detection
* 🗺️ Interactive pharmacy map
* ⚡ Live stock updates
* 🛒 Cart & purchase system
* 👨‍⚕️ Admin inventory management

---

## ✨ Key Features

### 👤 User Features

* Search medicines with **autocomplete**
* Detect location using GPS / manual input
* View nearby pharmacies with distance
* Emergency mode → fastest result
* Add medicines to cart & purchase

### 🏪 Admin Features

* Secure login system
* Add/update medicine stock
* Manage pharmacy details
* Real-time data synchronization

## 🏗️ System Architecture

The system follows a **client-server model**:

* Frontend → UI + Map
* Backend → Logic + API
* Database → Pharmacy + Medicine data
* APIs → Location + Map services

📌 The architecture diagram (page 11) shows data flow between:

* User → Frontend → Backend → Database → Map API 

---

## 🔄 Working Principle

1. User enters medicine name
2. System detects location (GPS/manual)
3. Nearby pharmacies are filtered
4. Availability is checked in database
5. Results shown in list + map
